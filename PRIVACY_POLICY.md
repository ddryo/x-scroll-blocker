# Privacy Policy / プライバシーポリシー

**X Scroll Blocker**

Last updated / 最終更新日: 2025-06-08

---

## 日本語

### 収集するデータ

この拡張機能は、個人情報やユーザーデータを一切収集、送信、共有しません。

### ローカルに保存するデータ

以下の設定情報を `chrome.storage.sync` を使用してブラウザ内にのみ保存します：

- スクロールブロック機能の有効/無効
- スクロール閾値の設定値
- 対象フィードの選択設定（検索ページ・プロフィールページ）

これらのデータはブラウザのストレージ同期機能を通じてユーザー自身のデバイス間でのみ同期され、開発者や第三者のサーバーには送信されません。

### 外部との通信

この拡張機能は外部サーバーとの通信を一切行いません。

### お問い合わせ

ご質問がある場合は、GitHub リポジトリの Issues よりご連絡ください。
https://github.com/ddryo/x-scroll-blocker/issues

## English

### Data Collection

This extension does not collect, transmit, or share any personal information or user data.

### Locally Stored Data

The following settings are stored only within your browser using `chrome.storage.sync`:

- Scroll blocking feature on/off status
- Scroll threshold value
- Target feed selection (search page, profile page)

This data is synced only across your own devices via Chrome's built-in storage sync feature. It is never sent to the developer or any third-party server.

### External Communication

This extension makes no network requests to any external server.
